Privacy Policy

Deep Navy Elegant Watch Face

Effective date: 23rd February 2026

1. Overview

Deep Navy Elegant Watch Face is a watch face application for Wear OS devices. This application does not collect, store, transmit, or share any personal data.

2. Data Usage

The watch face displays the following information directly on the device:

  • Current time and date
  • Day of the week
  • Device battery level
  • Step count (provided by the system)

Step count and other information are accessed via system APIs provided by Wear OS and remain on the device. The application does not store this data and does not transmit it to any external servers.

3. Data Collection

This application does not:

  • Collect personal information
  • Store user data
  • Share data with third parties
  • Use analytics tools
  • Use advertising SDKs

All displayed information is processed locally on the device.

4. Permissions

The application requests the following permission:

  • Activity Recognition – used solely to display step count on the watch face.

This permission is used only for on-device display purposes and no data is collected or transmitted.

5. Third-Party Services

This application does not use third-party analytics, advertising services, or external servers.

6. Children’s Privacy

The application does not knowingly collect personal data from anyone, including children.

7. Changes to This Privacy Policy

If this Privacy Policy is updated, the revised version will be made available at the same URL.
